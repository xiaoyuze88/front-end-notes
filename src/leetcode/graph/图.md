# 图

## 定义

图由顶点和边组成，顶点有权重（也称成本）。

如果图的顶点对是有序的，则称有向图，对顶点按权重排序后，便可以在顶点直接画一个箭头

## 领接表

图的实际信息保存在边上，因为它描述了图的结构，我们将描述边的方法称为`领接表`，或`领接表数组`，如：

```
[[2], [2], [0, 1, 3, 4], [2], [2]]
```
便描述了一个图，其中数字为各顶点的下标

## 领接矩阵

M[i, j] = 1 代表 i => j 有线，带权 的图 则 1 换成权重

## 深度优先搜索（dfs）

可以判断是否是连通图

## 广度优先搜索（bfs）

可以判断两个节点直接是否有路径，还可以找出这两个节点的最短路径
